# (Ebay) Kleinanzeigen - Anzeige duplizieren

Ein Tampermonkey-Userscript, das eine **„Anzeige duplizieren"**-Funktion auf kleinanzeigen.de hinzufügt.

## Features

- Fügt einen **„Anzeige duplizieren"**-Button direkt neben dem Speichern-Button ein
- Funktioniert mit dem **aktuellen React-basierten Layout** von kleinanzeigen.de
- Kein jQuery, keine externen Abhängigkeiten

## Installation

1. [Tampermonkey](https://www.tampermonkey.net/) für deinen Browser installieren
2. [Hier klicken um das Script zu installieren](https://github.com/Streammet/DEIN-REPO-NAME/raw/main/kleinanzeigen-duplizieren.user.js)

## Verwendung

1. Öffne eine deiner Anzeigen auf kleinanzeigen.de und klicke auf **„Bearbeiten"**
2. Neben dem „Anzeige speichern"-Button erscheint jetzt ein grüner **„Anzeige duplizieren"**-Button
3. Klick drauf – die Anzeige wird als neue Kopie eingestellt, das Original bleibt erhalten
